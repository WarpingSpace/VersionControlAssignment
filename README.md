
    What is Version Control and why is it used?
    It is a system to track changes to files.
    
    What's the difference between Git and GitHub?
    Git: a tool that let you send your work to the cloud to be saved there.
    GitHub: it is a website that store your work free and other can collaborate too.
    
    Describe what each of the following commands do in your own words:
        Git Status = shows the current state of your files.
        Git Add = tells git which changes to keep.
        Git Commit -m "" = commits a note describing your changes.
        Git Push = used to save your work to the cloud.
